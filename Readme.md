# Flask 入门实践

**目标**:开发一个简单的Watchlist程序

## 准备工作

### 使用git

`git init`

`vim .gitignore`

`git remote add origin git@github.com:...`

### 创建虚拟环境

**venv模块**

创建`python3 -m venv env`

激活`. env/bin/activate`

退出`deactivate`

### 安装Flask

`pip install flask`

更新`pip install -U flask` 

### 提交改动

```bash
git add .
git commit -m 'Ready'
git push -u origin main
```
